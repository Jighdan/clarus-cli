## Using Pip
```bash
	$ pip install notew-jighdan
```

## Manual
```bash
	$ git clone https://github.com/Jighdan/notew-cli
	$ cd notew-cli
	$ python setup.py install
```

# Usage

```bash
	$ notew
	
	# adds a new note
	$ notew -n "YOUR NOTE HERE"
	
	# (X Represents the note index)
	# deletes a note
	$ notew -d X
	# updates a note content
	$ notew -u X "NEW CONTENT HERE"
```
